Finder 에서는 시스템 파일이나 `.` 으로 시작하는 파일들을 기본적으로 숨겨놓는다.  
가끔 그런 파일에 접근해야할 때가 있는데,  
`shift + command + .` 키를 누르면 간단하게 표시/숨김 을 토글할 수 있다.
